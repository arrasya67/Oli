# Oli
Progressive Web App (PWA) untuk monitoring perawatan motor dan visualisasi tren pengeluaran servis. Built with Tailwind CSS, Alpine.js, and Chart.js.
###  Fitur Unggulan:
Offline-First Capabilities: Dapat dibuka, digunakan, dan menginput data di area tanpa sinyal internet berkat sistem Service Worker Caching.
Smart Odometer Tracking: Progress bar pemakaian oli interaktif berdasarkan kalkulasi sisa kilometer menuju target ganti oli berikutnya.
Statistik & Analisis Pintar: Visualisasi data pengeluaran (Tren Bulanan & Porsi Kategori) secara *real-time* menggunakan Chart.js.
Dinamis Dark Mode Perpindahan tema gelap dan terang yang nyaman di mata dan otomatis menyesuaikan preferensi sistem.
Local Storage Persistence: Seluruh data riwayat disimpan dengan aman langsung di dalam memori internal browser perangkat pengguna.

### Tech Stack:
UI/Styling:** Tailwind CSS & Plus Jakarta Sans Fonts
Reactivity Framework: Alpine.js
Data Visualization: Chart.js
PWA Core: Service Worker (`sw.js`) & Web App Manifest (`manifest.json`)
